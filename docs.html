Всё что нужно знать о Elancag

Slash-команды, голосовые утилиты, каналы-статусники — подробный гайд с примерами использования.

🚀 Быстрый старт

Бот уже работает на сервере — никакой установки не нужно. Просто начни вводить / в любом текстовом канале и Discord покажет список доступных команд.

💡
Если команды не появляются — попробуй перезайти в Discord или обнови приложение.

⌨️ Как вызвать бота

Есть два способа: slash-команды и префиксные команды.

Slash-команды (рекомендуется)

Введи / в чате, выбери нужную команду из меню. Discord автоматически покажет параметры и подсказки.

Префиксные команды

Пиши команду с префиксом $$ прямо в чат. Сообщение и ответ бота удалятся автоматически через несколько секунд, чтобы не засорять канал.

СпособПримерОсобенности
Slash /ping Подсказки, автодополнение, рекомендуется
Префикс $$ping Ответ удаляется через ~8 сек
Алиас $$sreg Короткий псевдоним для /setregion
⚠️
Некоторые команды с параметрами (например /setstatus) работают только через slash — префиксный вариант недоступен.

🛠️ Команды

🏓
/ping
Проверить задержку бота до серверов Discord
кд: 5 сек виден только тебе
⌄

Показывает два значения задержки: Roundtrip — время от отправки до получения ответа, и WebSocket — постоянное соединение с Discord. Цвет embed зависит от качества пинга.

Использование
SLASH
/ping
PREFIX
$$ping
Пример ответа
@user → /ping
🏓 Понг!
> Roundtrip: `42ms`
> WebSocket: `67ms`
🟢
Зелёный — менее 100 мс, жёлтый — до 250 мс, красный — больше 250 мс.
🤖
/info
Информация о боте: аптайм, серверов, пользователей, версия
кд: 15 сек
⌄
Использование
SLASH
/info
PREFIX
$$info
Пример ответа
@user → /info
🤖 Информация о боте
> Аптайм: `3д 14ч 22м 8с`
> Серверов: `12`
> Пользователей: `3842`
> Пинг WS: `67ms`
> Node.js: `v22.5.0`
📖
/help
Интерактивная справка по всем командам бота
кд: 5 сек виден только тебе
⌄

Открывает embed со списком всех команд. Через выпадающий список можно выбрать конкретную команду и получить подробное описание — способы вызова, права, ограничения.

Если команда отключена на сервере или у тебя нет к ней доступа — это будет видно прямо в справке.

Использование
SLASH
/help
PREFIX
$$help
💡
Команды с пометкой 🔒 ограничены для тебя. Команды в зачёркнутом тексте отключены на этом сервере.
🌍
/setregion
Изменить регион голосового канала, в котором ты находишься
Manage Channels кд: 10 сек
⌄

Меняет RTC-регион голосового канала, в котором ты сидишь в момент вызова команды. Это влияет на качество связи — выбери ближайший к тебе регион.

Использование
SLASH
/setregion — следующий регион по кругу
SLASH
/setregion регион:rotterdam — конкретный регион
PREFIX
$$sreg — следующий по кругу (быстрый алиас)
Смена на конкретный регион
@user → /setregion регион:rotterdam
🌍 Регион изменён
> Канал: #Войс-1
> Был: `🌐 Automatic`
> Стал: `🇳🇱 Rotterdam`
> Участников: `4`
> Сообщение удалится через 8с
Быстрая смена через алиас
@user → $$sreg (находясь в голосовом канале)
🌍 Регион изменён
> Был: `🇳🇱 Rotterdam` → Стал: `🇸🇬 Singapore`
> Сообщение удалится через 8с
Ошибка — не в канале
@user → /setregion (без голосового канала)
🔇 Ты не в голосовом канале
Зайди в голосовой канал и повтори команду.
ℹ️
Ответ удаляется через 8 секунд, чтобы не засорять канал. Смена региона происходит мгновенно — все участники ощутят изменение сразу.
💬
/setstatus
Установить или очистить статус голосового канала
Manage Channels кд: 10 сек
⌄

Устанавливает текстовый статус голосового канала — эта строка отображается под именем канала и видна всем участникам сервера. Можно установить любой текст или очистить статус.

Параметры
ПараметрОбязательныйОписание
канал✅ ДаГолосовой или Stage-канал
статус❌ НетТекст статуса. Если не указан — статус очищается. Максимум 500 символов.
Использование
SLASH
/setstatus канал:#Войс статус:🎮 играем в CS2
SLASH
/setstatus канал:#Войс — (без статуса, чтобы очистить)
Установка статуса
@admin → /setstatus канал:#Войс-1 статус:🎮 играем в CS2
📝 Статус установлен
> Канал: #Войс-1
> Статус: 🎮 играем в CS2
> Сообщение удалится через 10с
Очистка статуса
@admin → /setstatus канал:#Войс-1 (без параметра статус)
🗑️ Статус очищен
> Канал: #Войс-1
> Статус канала очищен
✨
/unic
Управление уникальными командами и особенностями сервера
Только администраторы кд: 5 сек виден только тебе
⌄

Позволяет администраторам просматривать и управлять уникальными командами и включаемыми особенностями (фичами) именно на вашем сервере.

Субкоманды
SLASH
/unic list — список всех уникальных команд и фич
SLASH
/unic info название — подробная информация о команде или фиче
SLASH
/unic enable фича — включить особенность
SLASH
/unic disable фича — выключить особенность
Просмотр списка
@admin → /unic list
✨ Уникальное — сервер
📋 Уникальные команды
> /rules — правила сервера

✨ Особенности сервера
> ✅ Игровые статусы · `game_status`
Включение фичи
@admin → /unic enable фича:game_status
✅ Игровые статусы включены
Отображает игровую активность участников в каналах.

> Фича активна и работает в реальном времени.
> Для отключения: /unic disable

🔒 Права и доступ

Некоторые команды ограничены по правам или доступны только определённым пользователям на сервере.

1
Глобальный бот-админ — имеет доступ ко всему на всех серверах, игнорирует кулдауны. Назначается при настройке бота.
2
Серверный бот-админ — те же права, но только на своём сервере. Назначается администратором через HTTP API или веб-панель.
3
Администратор Discord — право Administrator даёт доступ к /unic. Право Manage Channels — к /setregion, /setstatus.
4
Особые роли/пользователи — администраторы могут выдать доступ к /setregion конкретным ролям или участникам через настройки сервера.
⚠️
Если видишь ошибку «🚫 Нет доступа» или «🔒 Команда отключена» — обратись к администратору сервера.

⏳ Кулдауны

У каждой команды есть кулдаун — минимальный интервал между вызовами. При попытке вызвать команду слишком быстро бот напомнит, сколько нужно подождать.

КомандаКулдаун
/ping5 секунд
/help5 секунд
/unic5 секунд
/setregion10 секунд
/setstatus10 секунд
/info15 секунд
Ответ при кулдауне
@user → /ping (слишком рано)
⏳ Подожди
Команда `/ping` будет доступна через 3.2с.
💡
Бот-админы не ограничены кулдаунами.
